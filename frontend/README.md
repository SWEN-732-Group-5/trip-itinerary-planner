# Frontend

This project is built using React + Vite
Please view the main README before exploring here

## Installation

### Required
- [Node.JS](https://nodejs.org/en)
- [pnpm](https://pnpm.io/)

## Available Scripts

Run the following options using `pnpm run <option>`

options:
- `dev` - Start vite dev server
- `build` - Create production build
- `lint` - Run ESLint
- `preview` - Preview the production build
